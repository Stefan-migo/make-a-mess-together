# Multi-Agent Requirements

Every task must genuinely require multiple sub-agents. Minimum 2 sub-agents per trajectory. Sub-agent creation calls must be self-contained (all IDs, dates, rules inline). Sub-agent outputs must be meaningfully used in final response. Coordinator must not duplicate sub-agent work. Patterns: Parallel Search, Parallel Analysis, Specialist Delegation, Productivity Flow, Parallel Generation.
