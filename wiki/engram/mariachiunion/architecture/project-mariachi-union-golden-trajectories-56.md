# Project: Mariachi Union — Golden Trajectories

Mariachi Union is a multi-agent OpenClaw project for creating and refining Golden Trajectories. Every task is single-turn: the prompt and a pre-written hint are sent together as the first message, the model works, and the contributor cleans and submits the resulting trajectory.
