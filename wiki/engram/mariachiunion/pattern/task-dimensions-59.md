# Task Dimensions

Every task defined by 4 dimensions: Type (Single Turn), Domain (Health/Exploration/Advice/Relationships/Time), Difficulty (Single-App/Multi-App Light/Multi-App Complex), Category (Memory Usage/Long-Horizon Context/Skill Discovery/Claw Native Tools/Skill Gap). Difficulty: Single-App=one domain, Multi-App Light=2 domains simple handoff, Multi-App Complex=3+ domains coordinated multi-step.
