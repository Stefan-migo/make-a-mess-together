# SpellingChime project structure and guidelines

**What**: Read both SpellingChime Documentation files (AttempterGuidelines.md - 209 lines, Course1.md - 172 lines). SpellingChime is a side project of MeterPavilion that generates conversation seeds (topic packs) for the main pipeline.

**Key Rules**:
- Session 1: Generate topic pack with 20-30 milestones using OpenClaw agent
- Session 2: Roleplay 40-50 turns of natural conversation following the topic pack
- Distractors (off-topic beats): Acceptable up to 10-20% of turns, NOT required
- If agent gives nonsense for 2-3 turns: Redirect to a different milestone/angle, never let 3+ turn loops form
- Always start fresh session for Session 2 (no carryover from Session 1)
- Stay in the universe — all questions must reference real data in the environment
