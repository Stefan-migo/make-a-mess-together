# SpellingChime Task0 - grace_lin universe database schema

**What**: Mapped the entire grace_lin PostgreSQL database schema for Spelling Chime Task0

**Why**: Understanding the universe data is the first step to generating a grounded topic pack

**Where**: SpellingChime/Tasks/Task0/database/dumpDatabase.json

**Learned**: 
- grace_lin is a PERSONAL HEALTH + FITNESS + FINANCE + PRODUCTIVITY universe
- 10 application schemas with ~50+ tables total
- Key domains: Apple Health (sleep/heart rate/steps/workouts), Strava (cycling/running), FinTrack (accounts/transactions/subscriptions), Calendar, Contacts, Email, Messaging, Notion, Sonos, Reminders
- Not in the 11 standard documented universes — it's custom but follows the same pattern
