# Git Branching Workflow Extension

Git repository initialization, feature branch creation, numbering (sequential/timestamp), validation, remote detection, and auto-commit for Spec Kit.

## Commands
| Command | Description |
|---------|-------------|
| `speckit.git.initialize` | Initialize a Git repository |
| `speckit.git.feature` | Create a feature branch |
| `speckit.git.validate` | Validate branch naming |
| `speckit.git.remote` | Detect Git remote URL |
| `speckit.git.commit` | Auto-commit changes |
